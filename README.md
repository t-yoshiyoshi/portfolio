# portfolio

## 概要
Webデザイナーとして制作したポートフォリオサイトです。  
デザインだけでなく、HTML / CSS / JavaScript を用いた  
アニメーション表現やユーザー体験を意識して制作しました。

▶ 公開URL  
https://t-yoshiyoshi.github.io/portfolio/

---

## 制作目的
- 自身のデザイン・フロントエンドスキルをまとめて紹介するため
- アニメーションやインタラクションを含めた表現力を伝えるため
- 実務を想定した構成・保守性を意識して制作するため

---

## 使用技術
- HTML5  
- CSS3  
- JavaScript
- GSAP（ScrollTrigger）

---

## 実装・工夫したポイント
- GSAP + ScrollTrigger を用いたスクロール連動アニメーション
- コンテンツが順番に表示されるUI設計
- 画像・動画を含めた表現
- GitHub Pages を利用した静的サイト公開
